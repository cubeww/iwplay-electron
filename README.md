# IWPLAY

Experimental "I Wanna Fangames" Management Software

## Development

vscode + typescript + electron-vite + vue3 + pinia

## Build

1. Install node.js

2. Download or clone the repository

3. Open the repository in vscode

4. Execute once:

   ```sh
   npm install
   ```

5. To run, execute:

   ```sh
   npm run dev
   ```
